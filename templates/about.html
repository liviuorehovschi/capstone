Project Motivation

Lung cancer remains a leading cause of cancer-related mortality worldwide. Early detection of lung adenocarcinoma and squamous cell carcinoma is crucial for improving patient outcomes. However, traditional diagnostic methods are time-intensive and subject to variability between pathologists.

This project aims to leverage advanced AI techniques, specifically Convolutional Neural Networks (CNNs), to assist pathologists by improving the speed, consistency, and accuracy of histopathological image analysis.

Project Goals

The goals of this project include:

  • Developing a CNN model capable of classifying lung tissue images into benign, adenocarcinoma, and squamous cell carcinoma categories.
  • Building a user-friendly web application to integrate the model into a practical diagnostic tool.
  • Addressing dataset limitations through robust data augmentation techniques.
  • Providing a scalable framework for future diagnostic applications in medical imaging.

Development Journey

Phase 1: Research

Conducted a literature review and gathered insights into the challenges of histopathological analysis.

Phase 2: Data Preparation

Processed the LC25000 dataset, including data augmentation, to ensure robust training and validation.

Phase 3: Model Development

Designed and trained the EfficientNet-B0 model with techniques like transfer learning and dropout regularization.

Phase 4: Web Application

Integrated the trained CNN model into a fully functional web application for real-time image classification.
