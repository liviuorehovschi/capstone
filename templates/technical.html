Model Architecture

CNN Architecture Overview

The EfficientNet-B0 model was chosen for its balance of high accuracy and computational efficiency. Key features include depth-wise separable convolutions and compound scaling, making it ideal for histopathological image analysis.

Key Components

  • Input Layer: 224x224 RGB images
  • Feature Extraction: EfficientNet convolutional blocks
  • Classification Head: Dense layers with softmax activation
  • Optimization: Adam optimizer with learning rate scheduling
